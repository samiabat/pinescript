# Quick Start Guide - Trading Session SAMI

Get your Trading Session SAMI indicator running in TradingView in 3 minutes!

## üöÄ 3-Minute Setup

### Step 1: Copy the Code (30 seconds)
1. Open `trading_session_sami.pine` file in this repository
2. Click "Raw" button (or open the file)
3. Select All (Ctrl+A or Cmd+A)
4. Copy (Ctrl+C or Cmd+C)

### Step 2: Add to TradingView (1 minute)
1. Go to https://www.tradingview.com
2. Log in to your account (create free account if needed)
3. Open any chart
4. Click **"Pine Editor"** tab at the bottom
5. Clear any existing code
6. Paste the code (Ctrl+V or Cmd+V)
7. Click **"Save"** button (name it: "Trading Session SAMI")
8. Click **"Add to Chart"** button

### Step 3: You're Done! (30 seconds)
- You should now see colored boxes on your chart
- Blue = Asia | Orange = London | Green = New York
- Session names appear at the top of each box

## üéØ What to Look For

### Asia Session (Blue)
- Forms the initial daily range
- Often becomes liquidity pool

### London Session (Orange)  
- Typically sweeps Asia high or low
- High volatility at London open

### New York Session (Green)
- Major directional moves
- Confirms or reverses London move

## üí° Quick Trading Tips

1. **Wait for session to close** before trading the range
2. **Watch for sweeps** above/below previous session
3. **Use session highs/lows** as support/resistance
4. **Avoid trading** during session transitions (first 15-30 min)
